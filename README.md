### All-in-One
#### Windows

1. 下载 [ShadowsocksR](http://omgib13x8.bkt.clouddn.com/ssr-win.7z)，解压并运行。

2. 右键`ShadowsocksR`托盘图标，点击`SSR服务器订阅 -> SSR服务器订阅设置`。

3. 找到 [](https://xn--nos809b.com/user) 的`All-in-One`模块，点击`复制订阅地址到剪切板`，填进`地址`，点击`Add`，点击`确定`（请确定左边列表中是否显示有“https://xn--nos809b.com/xxxxxx“，如果没有请再次添加）。

4. 右键`ShadowsocksR`托盘图标，`服务器 -> 更新SSR服务器订阅（不通过代理）`。

5. 选择一个合适的节点，系统代理模式选`全局模式`，代理规则选`绕过局域网和大陆`。

##### 游戏模式

1. 下载 [ShadowsocksR-Tap](http://omgib13x8.bkt.clouddn.com/ssr-tap.7z)，解压并运行。

2. 右键`ShadowsocksR`托盘图标，选择`订阅管理`，将订阅地址添加进去即可。

#### macOS

1. 下载 [ShadowsocksX](http://omgib13x8.bkt.clouddn.com/ssr-mac.dmg)，拖至`应用程序`文件夹内，运行。

2. 点击`ShadowsocksR`托盘图标，点击`服务器 -> 编辑订阅`。

3. 点击`+`，找到 [墙洞](https://xn--nos809b.com/user) 的`All-in-One`模块，点击`复制订阅地址到剪切板`，填进`订阅地址`，点击`OK`。

4. 点击`服务器 -> 手动更新订阅`。

5. 选择一个合适的节点，选择`Pac/白名单模式`。
